DigIt Video, DigIt keyframes, and Dig It Video Keyframes all refer to Dig It, the local-first app for extracting useful still images from video.

Why this page exists

Some people search for the app name as DigIt without a space. This page helps search engines connect that spelling with Dig It and the main video keyframe extraction workflow.

What Dig It does

Import a local video, review a cleaner set of keyframes, choose covers or step images, and export selected PNG files.
